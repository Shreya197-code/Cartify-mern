# Cartify Backend

Cartify Backend is a Node.js and Express.js based REST API for an e-commerce application. It provides authentication, user management, product management, order processing, and payment integration.

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcryptjs
* Cloudinary
* Multer
* Razorpay
* Nodemailer

## Features

* User Registration
* User Login & Logout
* Hash the password
* OTP send and verification for email confirmation
* Welcome Mail
* JWT Authentication
* Role-Based Access Control (User/Admin)
* Product Management
* Image Upload with Cloudinary
* Order Management
* Razorpay Payment Integration
* Email Notifications
* Product controller
* Database for products
